Mouseenter/Mouseleave

Introduction

This demo shows how to use the custom mouseenter and mouseleave events. Just hover the grey box below.

Why? MooTools features mouseenter and mouseleave because mouseover/mouseout sometimes just does not work as expected. Mouseenter only fires once you enter the element and does not fire again if your mouse crosses over children of the element.

Menu Example

This example explains how to open a menu-like element on mouseenter and how it closes again on mouseleave.
